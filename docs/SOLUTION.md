# Описание решения

## Математическая идея

Пусть измеренный дискретный сигнал имеет вид:

```text
x[n], n = 0, 1, ..., N - 1
```

где:
- `N = 100000`;
- интервал дискретизации `Δt = 0.1 ms = 0.0001 s`;
- частота дискретизации `f_s = 1 / Δt = 10000 Hz`.

Частотная сетка для быстрого преобразования Фурье определяется формулой:

```text
f_k = k / (N * Δt)
```

для бинов дискретного преобразования Фурье.

Алгоритм решения:
1. Загрузить бинарный сигнал из файла.
2. Удалить среднее значение:
   ```text
   x_centered[n] = x[n] - mean(x)
   ```
3. Применить окно Ханна для уменьшения спектральных утечек.
4. Вычислить одностороннее БПФ:
   ```text
   X[k] = rFFT(x_centered[n])
   ```
5. Построить амплитудный спектр.
6. Оценить уровень шума как медиану амплитуды спектра.
7. Найти локальные максимумы с амплитудой не меньше `2 * noise_floor`.
8. Сохранить только бины вокруг найденных пиков.
9. Выполнить обратное БПФ и восстановить отфильтрованный сигнал.

## Почему метод работает

Случайный шум распределяется по большому числу частот, а полезный сигнал создаёт выраженные пики в амплитудном спектре.

По условию задачи амплитуды гармоник полезного сигнала как минимум в два раза больше амплитуд гармоник шума. Поэтому выделение пиков в частотной области и последующая фильтрация позволяют отделить полезный сигнал от шумовой составляющей.

## Практические предположения

Проект рассчитан на следующие условия:
- в файле ровно `100000` отсчётов;
- интервал дискретизации равен `0.1 ms`;
- бинарный формат данных относится к одному из типов: `float32`, `float64`, `int16`, `int32`, `int64`;
- по умолчанию используется порядок байтов little-endian.

## Как использовать проект 

1. Поместить бинарный файл своего варианта в папку проекта.
2. Запустить программу командой:
   ```bash
   python main.py --input path/to/your_variant_file.bin --output output
   ```
3. Если автоматическое определение типа данных не сработает, указать тип вручную, например:
   ```bash
   python main.py --input path/to/your_variant_file.bin --output output --dtype float32
   ```
4. После выполнения открыть файлы:
   - `output/detected_frequencies.txt`;
   - `output/filtered_signal.png`;
   - `output/spectrum.png`.
